Introduction
------------

Column Wizard allows you to make column style grids(and shapes) easily.



Install
-------

  1. Copy ColumnWizard.jsx to your Photoshop folder.

    Windows:
      <drive>:\Program Files\Adobe Photoshop <version>\Presets\Scripts\

    Mac:
      <drive>/Application/Adobe Photoshop <version>/Presets/Scripts/

  2. Restart your Photoshop.



Uninstall
---------

  1. Remove ColumnWizard.jsx

  2. Restart your Photoshop.



Usage
-----

  1. Run your Photoshop.
  2. Choose File -> Scripts -> ColumnWizard

  Tutorial:
    http://youtube.com/watch?v=HHqWL8xEWFA



Customize
---------

  You may modify the number of columns etc.

  1. See ColumnWizard.jsx and Modify the lines from SETTING BEGIN to SETTING END
